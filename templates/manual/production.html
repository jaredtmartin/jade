    {% trans "Design: Creating a Process" %}

    {% trans "The first step to do in production is to design processes. These are like recipies, but they don't only include the ingredients but also the end product. Suppose we were to make lemonade. We would need to create the following items: Lemons, Sugar, Water, and don't forget the end product:Lemonade. Click on Production, Process, Design. The create process window appears. Choose a document number for the process. This number will also be used to identify production jobs done based on this process. Add four lines, one for each of the four demo items we created. Now close the window and look at the lines we made. Suppose it takes 1 lemon, 2 scoops of sugar and 5 cups of water to make 1 serving of Lemonade. Since our inventory of lemons will go down when we make the lemonade, put -1 for the quantity of the lemons. Likewise put -2 and -5 respectively for the sugar and water. The last line, lemonade is our end product so put a positve 1 for its quantity. Save each line. There, now we have a process. Any time we want to make lemonade, Jade knows how much to use of each item." %}

    {% trans "Planning: Creating a Job" %}

    {% trans "Now suppose that we want to actually make some lemonade. But we don't want to make one serving, we want to make two servings. Click on Plan. When we click on plan, Jade collects all of the processes on the screen and creates jobs to match them. So be sure that once you have several processes that you click on the document number of the process you wish to execute before you click on Plan. Jade asks us how many times we want to do the process. Type 2 to plan for 2 servings and click Create. Jade will duplicate each line of the process into jobs appending a 1 to the end of the document number. If we were to plan this process again, it would append 2 and so on. Notice that the quantities of the items have been doubled. this is because we told Jade we wanted 2 times the Process. If we wanted to plan the job in the future we could change the date at this point, but for now we'll leave it as it is." %}

    {% trans "Execution: Starting a Job" %}

    {% trans "Once you're actually taking the supplies of the shelf, click on the document number of the job and click on Start. Notice that all of the lines with negative quantities are now activated. If we check the stock levels, the items used in the production of these items are not included in the inventory." %}

    {% trans "Execution: Finishing a Job" %}

    {% trans "Before we mark the job as finished, suppose we spilled some lemonade on the floor while we were working. How will we report this in the inventory? Change the quantity for Lemonade from 2 to 1.5 and save. Now when we click on Finish, one and a half servings of lemonade will appear in our inventory, ready to sell." %}
